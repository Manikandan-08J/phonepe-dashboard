# PhonePe Transaction Insights Dashboard

## Overview
This project analyzes PhonePe transaction data to uncover insights into user behavior, transaction patterns, and geographical trends across India.  
An interactive dashboard is built using Streamlit to visualize key metrics and support data-driven decision making.

---

## Objectives
- Analyze digital transaction data across states and districts
- Identify top-performing regions and categories
- Visualize insights using charts and maps
- Build an interactive dashboard for exploration

---

## Tech Stack
- **Python**
- **Pandas**
- **MySQL**
- **Streamlit**
- **Plotly**

## Features

### 🔹 User Engagement
